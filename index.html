pynipper-ng

Configuration security analyzer for network devices.

More info about pynipper-ng

Check the vulnerabilities and misconfigurations of routers, firewalls and switches reporting the issues in a simple way.

This tool is based on nipper-ng, updated and translated to Python. The project wants to improve the set of rules that detect security misconfigurations of the network devices using multiple standard benchmarks (like CIS Benchmark) and integrating the tool with APIs (like PSIRT Cisco API) to scan known vulnerabilities.

pynipper-ng is a free and opensource tool available in this repository

Get Started!

Goals

Check network device vulnerabilities

Check the vulnerabilities related with the device's SO version grouped by severity.

Detect device's misconfigurations

Detect configurations that allows attackers to exploit your network.

Easy to install and use

With only a CLI command you can check your device.

Free and open-source

Designed to be free and open-source!




Install Use


Install

To install pynipper-ng is necessary:

  • Python 3 (>=3.6)
  • pip

Download the latest release attached wheel.

After download the package, you should run pip install [package-name]



Use

Finally, you can check the CLI help with pynipper-ng -h




Start with pynipper-ng!

Clone, install and hack!
